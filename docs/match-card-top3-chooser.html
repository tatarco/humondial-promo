A — Balanced columns + gold puck VS

Clearest “two sides” duel; puck reads as pairing, not a score.

Pros

  • Familiar duel layout — easy RTL scan with a stable center anchor.
  • Distinct from numeric score semantics (gold circle ≠ score gutter).
  • Each team sits in its own lane (works if you later need per‑side taps).

Cons

  • Long club names wrap awkwardly unless you tighten line clamp / ellipsis.
  • A large puck steals width; tune size so flag + title stay comfy on ~360px.

B — Pitch split + glowing center stripe

Territory metaphor; spine is “halfway line,” not a score column.

Pros

  • Instant “two halves” rivalry read at a glance.
  • Stripe reads as divider / field chrome if glow stays restrained.
  • Visually distinctive without adding numbers.

Cons

  • Heaviest option — easy to crowd badges, meta, and CTAs.
  • If the glow is too strong, it can feel like a “scores lane” next to live UI.

C — Vertical gold spine + mirrored blocks

Portrait‑friendly symmetry; center column holds VS without digits.

Pros

  • Strong symmetry — quick parity check for crest + name balance.
  • Scales well on tall narrow phones.
  • Vertical spine is clearly decorative, not a score gutter.

Cons

  • Bright spine can compete with badges / primary CTA unless dimmed in calm states.
  • Long names need tight hierarchy so the beam doesn’t “cut” a block awkwardly.
